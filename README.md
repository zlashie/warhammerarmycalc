# Warhammer Army Strategic Analyzer

### Project Pitch
When constructing a **2000p** Warhammer 40k **10th edition** army, it's easy to calculate the expected statistical average damage output of it's individual units, but what is the total army output given all activations? This web application does statistical analysis of the entire army to attempt to give a hollistic overview of the strengths and weaknesses of your 2000p army, atleast from a damage output perspective.

### Quick Start
1. **Backend**: Navigate to `/backend`, run `./gradlew bootRun` (Java 25 required).
2. **Frontend**: Navigate to `/frontend`, run `npm install` then `ng serve`.

### Tech Stack
- **Backend**: Spring Boot 3.4+, Java 25
- **Frontend**: Angular 19+, Signals, TailwindCSS (optional)
