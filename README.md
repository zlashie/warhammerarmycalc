# Warhammer Army Strategic Analyzer

### Quick Start
1. **Backend**: Navigate to `/backend`, run `./gradlew bootRun` (Java 25 required).
2. **Frontend**: Navigate to `/frontend`, run `npm install` then `ng serve`.

### Tech Stack
- **Backend**: Spring Boot 3.4+, Java 25
- **Frontend**: Angular 19+, Signals, TailwindCSS (optional)
